# daria
Darian Calendar Dates for CLI

# Usage
`daria`

## Specify date
`daria 2019-01-02`

## Format date
`daria -f "Today is %s"`

### formatting options

* %Y - Mars Year
* %N - Mars Month
* %m - Mars Month Name
* %D - Mars Day
* %H - Mars Hour
* %M - Mars Minute
* %S - Mars Second
* %s - Mars Sol Name
